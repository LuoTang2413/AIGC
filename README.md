# AIGC

AIGC（AI-Generated Content）指人工智能生成内容。传统的UGC/PGC/OGC三者之间主要区别在于作者的利益关系、专业程度等，本质都是由人来产生内容，而AIGC则是由AI来产出内容，包括图片、文章、交谈、评论、视频、音频等等。
